# StaffServerKaspersky

Это клиент серверное приложение работающее по REST API.

----

## Требования

Основной датакласс это сотрудник у которого есть:

```python
@dataclass
class Employee:
    id: int
    name: str
    position: Optional[str] = None
    phone: str
    email: str
    age: int
    sex: str
```


#### Роуты:

GET запрос на получение списка сотрудников
POST запрос на создание сотрудника
PUT запрос на изменение сотрудника
DELETE запрос на удаление сотрудника
GET запрос на получение сотрудника по id


#### Тесты:

Тесты на получение списка сотрудников
Тесты на создание сотрудника
Тесты на изменение сотрудника
Тесты на удаление сотрудника
Тесты на получение сотрудника по id


#### Стэк:
* python3.10
* fastapi
* sqlite
* pytest
* не используй pydantic и sqlalchemy. Вместо них используй обычные датаклассы и сырые запросы к базе данных
